• Position(): position Means where is the elemet in the document like position from top and left . position() works according to parent element(relative element) and provides the position according to that element. it dosent include margin and padding while returning the content/elemnt position

  • Offset(): offset works according to document, means offset take the html document position and gives the position of element according to its main(body) element.offset returns the position included with gaps of paddings and margins.To set the position use $(".box").offset({ top: 0, left: 0 });

    • Note : With position and offset method we can both set and get the position of element in the document
