there are methods in jqury methods by which we can set or get the scroll value of windows or any div or element

  • scrollTop() : it returns the scroll value of scrollbar from top to its current position
  • scrollLeft() : it returns the scroll value of scrollbar from left to its current position
these can be used in any div or element and can be used for both setting or getting the scroll value
