In jquery there are preDefined Methods for Getting and Setting the Width and Height of Element :- Not only height and width but setting the height & Width of each segement of container

width() height()
InnerWidth() InnerHeight()
outerWidth() outerHeight()
outerWidth(true) outerHeight(true)
  • width() : this method is used for both setting and getting the width of any element content or we can use to set width of any element

  • innerWidth() : this method is used for getting and setting the inner width of any element

  • outerWidth() : this is used to get the outer width of any element with padding and border

  • outerWidth(true) : this is used get the whole widh of container including the padding and border and margin
